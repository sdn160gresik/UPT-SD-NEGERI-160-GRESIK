üí° A. Pengantar

Microsoft Word adalah program pengolah kata yang digunakan untuk mengetik, mengedit, dan mencetak dokumen seperti surat, laporan, dan tugas sekolah.

Sebelum kita bisa menggunakan Microsoft Word, kita harus membuka atau masuk ke program tersebut terlebih dahulu.

üñ•Ô∏è B. Langkah-langkah Masuk ke Microsoft Word

Berikut langkah-langkahnya secara berurutan:

  1. Nyalakan Komputer atau Laptop.
    Pastikan perangkat sudah hidup dan siap digunakan.

  2. Masuk ke Halaman Utama (Desktop).
    Setelah komputer menyala, kamu akan melihat layar utama yang disebut Desktop.

  3. Cari Ikon Microsoft Word.
    Biasanya ikon Word berwarna biru dengan huruf W.

    Kamu bisa menemukannya di:
    • Menu Start, atau
    • Di Desktop, jika sudah ada pintasannya.

  4. Klik Ikon Microsoft Word.
    Klik dua kali (double-click) pada ikon Microsoft Word untuk membukanya.

  5. Tunggu Program Terbuka.
    Setelah diklik, tunggu beberapa saat hingga jendela Microsoft Word muncul.

  6. Pilih Dokumen Kosong (Blank Document).
    Pada tampilan awal, pilih Blank Document agar kamu bisa mulai mengetik dokumen baru.

üìù C. Tampilan Awal Microsoft Word

Setelah terbuka, kamu akan melihat:

  • Area kosong putih tempat kamu mengetik, disebut lembar kerja.
  • Menu dan toolbar di bagian atas seperti File, Home, Insert, Page Layout, dan lainnya.

üéØ D. Ringkasan

Langkah-langkah membuka Microsoft Word:

  1. Nyalakan komputer
  2. Buka menu Start
  3. Pilih Microsoft Word
  4. Klik dua kali
  5. Pilih Blank Document
