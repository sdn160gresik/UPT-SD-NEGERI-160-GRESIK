üìò A. Pengertian Microsoft Word

Microsoft Word adalah program pengolah kata (word processor) yang digunakan untuk mengetik, mengedit, dan mencetak dokumen seperti surat, laporan, tabel, atau karangan.

ü™ü 1. Melalui Tombol Start

  1. Klik tombol Start (ikon Windows) di pojok kiri bawah layar.
  2. Pilih menu All Programs atau Semua Program.
  3. Cari dan klik folder Microsoft Office.
  4. Pilih Microsoft Word (misalnya Microsoft Word 2016).
  5. Tunggu hingga jendela Microsoft Word terbuka.

üíª 2. Melalui Shortcut di Desktop

  1. Cari ikon Microsoft Word di layar desktop.
  2. Klik dua kali (double-click) pada ikon tersebut.
  3. Microsoft Word akan terbuka dan siap digunakan.

‚å®Ô∏è 3. Melalui Pencarian (Search)

  1. Klik tombol Search (ikon kaca pembesar) di taskbar.
  2. Ketik Word di kolom pencarian.
  3. Klik Microsoft Word pada hasil pencarian.
  4. Tunggu sampai jendela Word terbuka.
üìÑ C. Tampilan Awal Microsoft Word

Saat pertama kali dibuka, Microsoft Word menampilkan:

  • Lembar kerja kosong (Blank Document) untuk mulai mengetik.
  • Menu Bar berisi tab seperti File, Home, Insert, Design, dan lainnya.
  • Ribbon berisi ikon-ikon perintah.
  • Area Pengetikan tempat menulis teks.
  • Status Bar di bagian bawah yang menampilkan informasi halaman dan jumlah kata.

  • Gunakan Ctrl + O untuk membuka dokumen lama.
üìö Kesimpulan

Menjalankan Microsoft Word dapat dilakukan dengan berbagai cara, yaitu:

  • Melalui Start Menu
  • Melalui Shortcut di Desktop

Microsoft Word digunakan untuk mengolah kata, membuat dokumen, dan mencetak hasil kerja.
